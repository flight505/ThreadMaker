ThreadMaker

Creates multi-start helical threads optimized for FDM 3D printing. Supports outer bumps on a cylinder, inner ridges in a bore, or matched pair mode with lug tab support.

Quick Start

  1. Build your jar neck and lid with a gap between them (0.5–1.0mm per side)
  2. Click Solid → flight505 → Create Thread
  3. Choose Matched Pair and select both faces (neck outer + lid bore)
  4. Adjust pitch, starts, and section size — check the info panel
  5. Click OK

Thread Modes

  • Outer (Male) — thread bumps on a cylinder
  • Inner (Female) — thread ridges inside a bore
  • Matched Pair — both at once with automatic engagement calculation

Female Styles (Matched Pair)

  • Full Thread — helical ridges inside the lid matching the male thread
  • Lug Tabs — simple curved tabs at the opening, easier to print and more tolerant

FDM Tips

  • Gap between neck and lid: 0.5–1.0mm per side (tutorial-proven values)
  • Thread thickness (section): 1.5–2.5mm circular works well
  • Use 0.12–0.16mm layer height in the thread zone
  • Pitch 3mm × 3 starts = quick-turn jar lid (closes in ~120° rotation)
  • Print jar upright, lid upside-down (flat surfaces on build plate)
